# Angular.js

**타입스크립트 기반으로 구성된 프레임 워크**



## Angular 명령어

- ngRepeat

  `<div ng-repeat="todo in todos">`

  todos리스트를 todo로 한 값씩 호출 -> for loop 처럼 사용

- ngFilter

  date  filter 사용 가능

- ngClick

   

  ```html
  <span class="input-group-btn">
   <button class="btn btn-default" type="button">Delete</button>
  </span>
  ```
